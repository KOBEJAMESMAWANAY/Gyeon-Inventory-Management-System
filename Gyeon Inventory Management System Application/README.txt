A Car Detailing Shop Inventory Management System (GYEON THEMED)
==============================================================================================

Description:
-----------
A GUI inventory manager for a car detailing shop (GYEON-themed) built with
Python and Tkinter. Tracks products by Product Number, Name, Category, and Quantity.
Provides Add, Update, Delete, Search, Sort, Category filter, and JSON persistence.
==============================================================================================

Features:
--------
- Add / Update / Delete items (product_number required)
- Realtime search (matches name or product number)
- Sort by Product Number, Name, Quantity, Category
- Filter by Category (modal Combobox)
- Save / Load / Export to JSON with summary of data stocks
- Live date/time system
